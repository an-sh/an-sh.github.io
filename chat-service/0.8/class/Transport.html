Class: Transport

Defined in: src/Transport.coffee

Overview

Transport public API interface. An instance of Transport implementation is available as a member of ChatService instance.

Instance Method Summary

  • # (null) emitToChannel(channel, messageName, messageData...) Sends a message to a channel (each room has a channel with the same name).
  • # (null) sendToChannel(id, channel, messageName, messageData...) Sends a message to a channel (each room has a channel with the same name), excluding the sender socket.
  • # (Object or null) getConnectionObject(id) Get an underlying connection object by id.

Instance Method Details

# (null) emitToChannel(channel, messageName, messageData...)

Sends a message to a channel (each room has a channel with the same name). This messages are sent directly, bypassing room history and permissions.

Parameters:

  • channel ( String ) — A channel.
  • messageName ( String ) — Message name.
  • messageData ( Rest... ) — Message data.

Returns:

  • null

# (null) sendToChannel(id, channel, messageName, messageData...)

Sends a message to a channel (each room has a channel with the same name), excluding the sender socket. This messages are sent directly, bypassing room history and permissions.

Parameters:

  • id ( String ) — Sender socket id.
  • channel ( String ) — A channel.
  • messageName ( String ) — Message name.
  • messageData ( Rest... ) — Message data.

Returns:

  • null

# (Object or null) getConnectionObject(id)

Get an underlying connection object by id.

Parameters:

  • id ( String ) — Socket id.

Returns:

  • ( Object or null ) — Connection object corresponding to the socket id. Returns null if the connection was closed.
