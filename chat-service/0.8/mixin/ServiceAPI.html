Method Details

~ (Promise<Array>) execUserCommand(context, command, args...)

Executes UserCommands.

Parameters:

  • context ( String or Boolean or Object ) — Is a userName if String, or a bypassPermissions if Boolean, or an options hash if Object.
  • command ( String ) — Command name.
  • args ( Rest... ) — Command arguments with an optional callback.

Options Hash: (context):

  • userName ( String ) — User name.
  • id ( String ) — Socket id, it is required for UserCommands#disconnect, UserCommands#roomJoin, UserCommands#roomLeave commands.
  • bypassHooks ( Boolean ) — If false executes command without before and after hooks, default is false.
  • bypassPermissions ( Boolean ) — If true executes command (except UserCommands#roomJoin) bypassing any permissions checking, default is false.

Returns:

  • ( Promise<Array> ) — Array of command results.
